# Raw Sources

This directory is the immutable source layer.

Codex may read these files during ingest and query workflows, but should not
rewrite them unless explicitly asked.

Suggested folders:

- `papers/`: PDFs, TeX files, bibliographies, paper notes.
- `notes/`: human notes, sketches, meeting notes, journals.
- `web/`: clipped web pages and saved articles.
- `data/`: datasets, result tables, configs, metadata.
- `assets/`: images and attachments.
