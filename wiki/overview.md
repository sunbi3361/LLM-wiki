# Overview

This wiki is an empty research memory scaffold.

Its purpose is to accumulate knowledge over time by compiling raw sources into
linked markdown pages. The initial scope is intentionally open: once the user
adds papers, notes, or web clips, Codex should ingest them into the appropriate
wiki pages and maintain this overview as the synthesis changes.

## Current State

- Source layer exists under `raw/`.
- Wiki layer exists under `wiki/`.
- Operating schema exists in `AGENTS.md`.
- Local skill instructions exist in `codex/skills/autosci-research-wiki/SKILL.md`.
