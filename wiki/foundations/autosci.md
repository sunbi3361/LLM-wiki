# AutoSci

Status: foundation

AutoSci is a memory-centric agentic system for scientific research. This wiki
borrows its lifecycle shape while keeping the local implementation markdown-first.

## Adapted Lifecycle

1. Build literature memory from sources.
2. Generate research ideas grounded in the wiki.
3. Design experiments for promising ideas.
4. Execute or document experiment runs.
5. Evaluate results with explicit verdicts.
6. Draft, review, and revise manuscripts.
7. Evolve the memory and operating procedures based on feedback.

## Local Mapping

- Literature memory: `wiki/papers/`, `wiki/concepts/`, `wiki/methods/`
- Active research memory: `wiki/ideas/`, `wiki/experiments/`, `wiki/manuscripts/`
- Review memory: `wiki/reviews/`
- Evolution record: `wiki/log.md`, `AGENTS.md`, `codex/skills/`
