# Command Menu

These are natural-language commands you can give Codex in this workspace.

## Setup and Status

- "현재 wiki 구조를 점검해줘"
- "wiki health check 해줘"
- "index와 log가 최신인지 확인해줘"
- "이 디렉토리를 Obsidian vault처럼 쓸 수 있게 정리해줘"

## Source Ingest

- "raw/papers의 새 논문을 ingest 해줘"
- "이 arXiv 논문을 raw/web에 기록하고 wiki에 요약해줘"
- "raw/notes의 메모를 읽고 관련 concept page를 만들어줘"
- "이 자료를 기존 wiki와 비교해서 contradiction을 찾아줘"

## Query and Synthesis

- "이 wiki 기준으로 [주제]를 설명해줘"
- "[A]와 [B]를 비교 표로 정리하고 wiki/questions에 저장해줘"
- "현재 연구 방향의 핵심 가설과 근거를 요약해줘"
- "읽어야 할 다음 논문 후보를 추천해줘"

## Ideation

- "현재 wiki에서 연구 아이디어 5개를 만들어줘"
- "가장 유망한 아이디어 하나를 novelty check 해줘"
- "아이디어를 critique하고 약점을 보완해줘"
- "이 아이디어를 wiki/ideas에 proposal 형식으로 저장해줘"

## Experiments

- "이 아이디어를 실험 계획으로 바꿔줘"
- "baseline, ablation, metric, dataset을 포함해서 실험 설계를 만들어줘"
- "실험 결과를 읽고 verdict를 내려줘"
- "결과를 wiki/experiments와 관련 idea page에 반영해줘"

## Writing

- "관련 연구 섹션 초안을 만들어줘"
- "paper plan을 작성해줘"
- "현재 idea와 experiment를 바탕으로 논문 outline을 만들어줘"
- "리뷰어 관점에서 초안을 평가해줘"
- "rebuttal 초안을 만들어줘"

## Maintenance and Evolution

- "중복 concept page를 합쳐줘"
- "broken wikilink를 찾아서 고쳐줘"
- "출처 없는 claim을 찾아줘"
- "AGENTS.md를 지금 쓰는 방식에 맞게 개선해줘"
- "이 wiki를 더 AutoSci 스타일로 진화시킬 다음 작업을 제안해줘"
